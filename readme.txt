A collection of useful tools and patches for WordPress sites.

== Description ==

Toolbox provides small utilities and patches that help improve and maintain multiple WordPress installations.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/toolbox` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
